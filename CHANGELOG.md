# Changelog

All notable changes to the DevTrackr JetBrains plugin will be documented in this file.

## [1.0.0] - 2025-01-25

### Added
- Initial release of DevTrackr for JetBrains IDEs
- Automatic time tracking for coding sessions
- Activity tracking for file changes and editor switches
- API key configuration in settings
- Idle detection and heartbeat functionality
- Git branch and repository URL detection
- Debug logging support
- Settings UI for API key management
- Menu action "DevTrackr - Set API Key" in Tools menu
- Support for IntelliJ IDEA, Rider, PyCharm, WebStorm, and other JetBrains IDEs
- Compatibility with IDE builds 233 to 999.* (future-proof)

### Features
- Tracks coding activity automatically
- Sends activity data to DevTrackr API
- Detects idle periods and stops tracking
- Integrates with Git repositories
- Lightweight and non-intrusive
